# Spybot
arduino and esp8266 based spying robot with android smartphone  
---------------------------------------------------------------------------------
# Specifications

* object recogination;
* face recogination
* alert message when weapon detected
* wireless android remote control
* gas detection and alert message
------------------------------------------------------------------------------------
# Hardwares
* arduino 
* nodemcu(esp8266)
* motor driver (l293n)
* smartphone (low end)
* chasis
* dc motors
* jumper wires
-------------------------------------------------------------------------------------
# Software
* arduino IDE
* google ML app
* linux os (prefered kali linux)
